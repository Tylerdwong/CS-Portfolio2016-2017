How to make a To do list

Step 1-HTML

My how to page needed javascript, html, and css in order to make it the best I can. For the HTML on a to do list it is important that you want to see how you want to format your to do list to make it look the way you want it to look. So what I did is that I added to style it the way I wanted the page to look with a dropdown button and a textbox to input my list of works, and even a submit button to make my to do list appear. After this you want to create your options which are basically 4 of them to make the input that you put in the textbox have a special effect which in this case I have my list being underlined, green font, blue font, or a red border around the list. In these options you want to make sure to id each of the options so later on you won't have to worry about not being organized. You would also want to name your div which is very important part to work in with the javascript, in this case my div name is "outputarea." Make sure all of this work is done in the body and lastly include your code to connect your html to your javascript so that it will operate in this case my javascript is named "todo.js".

Step 2-JavaScript

First you want to create your variables so that when you are coding you can make sure you have variables that are labeled and that will operate. Under the variables you want to add a code that will make your submitbutton operate with the code "addEventListener." In order to have your button output an answer you will have to use div + div + whatever you want to name your textbox where you put your input in. Then you have to make sure that you have "if statements" in order to have your outputs operate work. In these "if" statements you need to label what type of special element you are going to use when you have your output shown. It's also helpful to leave comments especially in your javascript so when you come back to it you will know what's going on and therefore be able to understand the sidenotes that are left there. And lastly have a code to delete things off of your to do list, and in this case to delete what task you want to get rid of on your list you will just click it and it will be gone.

Step 3-CSS

After you have finished all of your javascript and html and made sure that your to do list is working, you can now make your to do list look nice with css. Some effects from css I used were as simple as making borders around text and just changing the color of the background just to make your website look nice and the best to your ability.
